## GNU nano
Nano is a really simple text editor.  
The only commands you really need to know to work with nano are:
- `<C-o>` `Ctrl + o` => Save the file (`<C-s>` `Ctrl + s`)
- `<C-x>` `Ctrl + x` => Close the editor (`Alt + f4`)
